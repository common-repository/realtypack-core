== Description ==
Realty Pack is a dynamic and fully customizable WordPress theme, made specifically for Real Estate professionals. 

This theme gives you all the tools required to create beautiful and professional real estate websites powered by WordPress without any technical knowledge. Realty Pack is a user friendly builder, using drag & drop methods and Elementorâ€™s Page Editor. It is easy to navigate and puts the power and control back in the hands of the website owner. It is integrated with the latest version of WordPress and makes use of all the new features like the advanced Customizer, Custom Post Formats, Responsive Design, and more. Realty Pack allows you to quickly and easily create templates for 4 different kinds of pages/elements and offers 40+ customizable widgets. 

With the theme dedicated only to the real estate industry, you will find tools like the Single Property or Single Agent builder, Property Widgets, Agency Post formats and more tools specific to real estate.

== Frequently Asked Questions ==


= Automatic installation (easiest way) =

log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type "RealtyPack Core" and click Search Plugins. Once you have found it you can install it by simply clicking "Install Now".


= Manual installation =

**Uploading in WordPress Dashboard**

1. Download `realty-pack-core.zip`
2. Navigate to the 'Add New' in the plugins dashboard
3. Navigate to the 'Upload' area
4. Select `realty-pack-core.zip` from your computer
5. Click 'Install Now'
6. Activate the plugin in the Plugin dashboard

**Using FTP**

1. Download `realty-pack-core.zip`
2. Extract the `realty-pack-core` directory to your computer
3. Upload the `realty-pack-core` directory to the `/wp-content/plugins/` directory
4. Activate the plugin in the Plugin dashboard

The WordPress codex contains [instructions on how to install a WordPress plugin](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).
